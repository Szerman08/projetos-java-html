# Projetos Java e HTML

Coleção de projetos e exercícios desenvolvidos com Java e HTML, incluindo interfaces web e sistemas de cadastro.

## Sobre

Este repositório contém projetos práticos que combinam lógica de programação com desenvolvimento de interfaces, utilizando HTML para front-end e Java para lógica.

## Tecnologias

- Java
- HTML5
- CSS3

## Projetos

| Projeto | Descrição |
|---|---|
| Calculadora de IMC | Calcula o Índice de Massa Corporal |
| Calculadora de Idade | Calcula a idade a partir da data de nascimento |
| Tela de Menu | Interface de navegação |
| Tela de Login | Formulário de autenticação |
| Cadastro de Clientes | Formulário de cadastro de clientes |
| Cadastro de Veículos | Formulário de cadastro de veículos |
| Cadastro de Serviços | Formulário de cadastro de serviços |

## Como usar

Abra os arquivos `.html` diretamente no navegador ou execute os arquivos Java com:

```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```
